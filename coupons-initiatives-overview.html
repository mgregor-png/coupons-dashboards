Why cashback is a content + impressions lever

Today, coupons pages only have content for merchants that provide promo codes. Cashback lets us create an offer for any retailer - even those with zero existing content. This generates new impressions across web (coupons pages), Genie (desktop), and Safari (mobile).

The primary value is registration uplift (gating cashback behind account creation), SEO defense (more pages with real offers = stronger organic presence), and extension exposure (more merchants in the extension = more activation surface). Direct margin is modest after payout costs - the real return is in the funnel it feeds.

Testing approach: Build a cashback offer type, collect user emails, run a small number of manual delivery offers to validate the mechanics before scaling.
