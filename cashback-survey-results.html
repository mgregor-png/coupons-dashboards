Cashback Survey Results

PostHog popover surveys on /coupons/, US traffic, all device types. Two consecutive runs combined.

Field dates: Apr 22 - Apr 27, 2026   |   Population: 28,379 unique impressions   |   Responses: 1,475 partial+complete

Most /coupons/ visitors already have the cashback habit and would consider a Groupon version. The opportunity is real and the audience is qualified.
  • Stable signal across two runs. Q1 distributions match within 1pp despite different time windows and a question-format change to Q3, so the population is consistent and the result is robust.
  • Capital One Shopping leads, Rakuten and Honey contested. Among users who named a specific tool, Capital One Shopping (~27%) is the most-mentioned, followed by RetailMeNot (~21%), Rakuten (~21%), and PayPal Honey (~16%).
  • Question format had no effect on intent. Single-choice (Yes definitely / Probably / Maybe / Probably not / No) and 1-5 rating scale produced statistically equivalent answers - 63% vs 61% top-2 positive intent.
  • Response rate ~5%, dismissal ~58%. Healthy for an unprompted on-page popover with no incentive.

Methodology and raw funnel

Two surveys, same target. Both ran as PostHog popovers triggered on URLs containing /coupons/, all device types, no audience segmentation. Run 1 ended when Run 2 launched, so they don't overlap.

Q3 format change. Run 1 asked "Would you use a Groupon cashback tool alongside what you use today?" with 5 named choices (Yes definitely / Probably / Maybe / Probably not / No). Run 2 replaced this with a 1-5 rating scale labeled "Definitely not" to "Definitely yes". Q1 and Q2 wording was identical.

Partial responses are enabled on both surveys, so each forward navigation fires a "survey sent" event. Q1 raw counts include these, which is why proportions are reported in this report rather than absolute numbers for Q1 and Q2.

Run 2 is still active at time of writing (Apr 27, 2026). Numbers will continue to accumulate. Top-line proportions are unlikely to shift materially - both runs are already well past the point where the distribution stabilises.

Q2 caveat. Multi-choice partial-response events make absolute counts unreliable. The relative ranking is stable across both runs (Capital One leads in both, Coupert is bottom-2 in both), so we use ranking rather than precise share.
